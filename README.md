# METU Orienteering, Google Formsdan OE2010 Formatına Dönüştürücü
Orienteering etkinlikleri için google formsdan OE2010 formatına dönüştürücü yazılım.

Programı kullanabilmek için internet bağlantınızın olması ve bilgisayarınızda "Python" yüklü olması gerekmektedir.
Python yüklemek için Windows Store'u kullanabilirsiniz.
